#### Installation
***
A little intro about the installation. 
```
$ git clone https://github.com/sam5711207/movies-sever-project.git
$ cd ../merval_actors
$ npm install
```
For execute the project you can run the following commands 
```nodemon index.js``` / ```node index.js```
